# 商家管理后台 - 掌故业务内嵌

## 项目背景
- 将掌柜业务线相关业务（不含报表）单独维护
- 前端使用React、Redux构建，项目入口：`/src/title.js`

## 注意
本项目使用[7秒部署](http://git.2dfire-inc.com/union/union-scripts)方案，提交前需要先在本地执行`npm run build`完成编译再提交。

## 项目使用

### 启动
```
npm run start
```

### 编译
```
npm run build
```

### 部署
```
npm run deplou
```

## 兼容性
- IE10、Edge、Chrome、Safari、Firefox
